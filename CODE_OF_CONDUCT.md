# Code of Conduct

## Our Pledge

This project is a technical collaboration space. Participants are expected to stay respectful, precise, and focused on the code, tests, documentation, and evidence.

## Expected Behavior

- Be constructive and factual.
- Keep feedback tied to code, contracts, tests, or documentation.
- Disagree on technical grounds without personal attacks.
- Respect maintainers, contributors, and users.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Deliberate sabotage, trolling, or abusive conduct.
- Repeated refusal to engage with documented project rules.

## Enforcement

Maintainers may close issues, lock threads, or decline contributions that violate this code of conduct or the project contribution rules.

## Reporting

Use the private security channel for security-sensitive reports, or open a normal issue for non-sensitive conduct concerns.
